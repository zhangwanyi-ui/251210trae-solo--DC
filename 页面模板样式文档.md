# DC在线业务系统 - 页面模板样式文档

## 1. 文档概述

### 1.1 文档目的

本文档基于"DC页面结构272页与对应模板_251021.xlsx"，详细描述了DC在线业务系统的页面模板样式，包括页面布局、核心组件、交互设计、样式规范等，为前端开发团队提供明确的页面样式实现依据，确保系统页面风格统一、布局合理、用户体验良好。

### 1.2 术语定义

| 术语 | 解释 |
|------|------|
| 左侧导航页 | 位于页面左侧的分类导航，包含5个分类的17个导航页 |
| 内容导航项 | 位于内容区域的导航，包含14个内容导航项，用于动态切换页面内容 |
| 功能入口导航项 | 位于顶部导航区的功能入口，包含5个功能入口的17个页面 |
| 页面模板 | 14个内容导航项对应的页面结构，用于快速构建272个页面 |
| 观察类模板 | 用于展示主体事实发生的页面模板，适用于"实时动态"、"近期发生"、"历史回顾"内容导航项 |
| 感知类模板 | 用于决策分析的页面模板，适用于"变化分析报告"、"对比分析报告"等5个内容导航项 |
| 关注类模板 | 用于主体关注的页面模板，适用于"我的个人关注"、"推荐的关注"、"强制关注"内容导航项 |
| 行动类模板 | 用于行动操作的页面模板，适用于"处理任务变更操作"、"响应操作"、"参与协同操作"内容导航项 |
| 详情类模板 | 用于展示详细信息的页面模板，适用于功能入口导航项及各页面详情 |

## 2. 页面模板总览

系统包含4种核心页面模板类型（基于DC页面结构272页与对应模板_251021.xlsx的模板分类），共14个内容导航项，用于快速构建272个页面：

| 模板类型 | 适用内容导航项 | 数量 | 占比 |
|----------|----------------|------|------|
| 观察类模板 | 实时动态、近期发生、历史回顾 | 3 | 21.43% |
| 感知类模板 | 变化分析报告、对比分析报告、分布特征分析报告、归因分析报告、相关性分析报告 | 5 | 35.71% |
| 关注类模板 | 我的个人关注、推荐的关注、强制关注 | 3 | 21.43% |
| 行动类模板 | 处理任务变更操作、响应操作、参与协同操作 | 3 | 21.43% |
| **总计** | - | **14** | **100%** |

## 3. 基础布局样式

### 3.1 三栏式布局

所有页面均采用统一的三栏式布局：

| 区域 | 宽度 | 高度 | 背景色 | 边框 | 阴影 |
|------|------|------|--------|------|------|
| 顶部导航栏 | 100% | 60px | #ffffff | 底部1px实线 #f0f0f0 | 0 2px 8px rgba(0, 0, 0, 0.08) |
| 左侧导航栏 | 240px (展开) / 80px (折叠) | 100vh - 60px | #001529 | 右侧1px实线 #f0f0f0 | 0 2px 8px rgba(0, 0, 0, 0.1) |
| 内容区域 | 剩余宽度 | 100vh - 60px | #f5f5f5 | 无 | 无 |

### 3.2 内容区域基本结构

所有页面模板的内容区域包含以下基本结构：

| 组件 | 高度 | 背景色 | 边框 | 内边距 | 外边距 |
|------|------|--------|------|--------|--------|
| 页面标题 | 60px | #ffffff | 底部1px实线 #f0f0f0 | 20px | 0 |
| 面包屑导航 | 40px | #ffffff | 无 | 0 20px | 0 |
| 内容导航 | 50px | #ffffff | 底部1px实线 #f0f0f0 | 0 20px | 0 |
| 页面内容 | 自适应 | #ffffff | 无 | 20px | 16px |

## 6. 响应式设计

### 6.1 桌面端（≥1200px）

- 完整三栏布局，左侧导航宽度240px
- 所有组件按设计稿正常显示
- 图表并排显示（监控页）

- 左侧导航可折叠，折叠后宽度80px
- 关键指标区每行2个卡片
- 图表垂直排列（监控页）
- 表单宽度调整为90%

### 6.3 移动端（<768px）

- 左侧导航隐藏，通过菜单按钮展开
- 关键指标区每行1个卡片
- 所有组件垂直排列
- 表单宽度100%
- 操作按钮区垂直排列
