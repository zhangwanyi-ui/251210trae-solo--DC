# DC在线业务系统 - 用户手册

## 1. 文档概述

### 1.1 文档目的
本用户手册旨在帮助用户了解DC在线业务系统的功能和使用方法，指导用户快速上手和高效使用系统。

### 1.2 适用范围
本手册适用于所有使用DC在线业务系统的用户，包括系统管理员、业务操作员、监控人员和决策人员。

### 1.3 术语定义
| 术语 | 解释 |
|------|------|
| DC | 业务系统的简称 |
| 顶部导航 | 位于页面顶部的功能导航，包含报警、事件、任务、在线、个人中心5个功能入口 |
| 左侧导航 | 位于页面左侧的分类导航，包含5个分类的17个导航页 |
| 内容导航 | 位于内容区域的导航，包含14个内容导航项，用于动态切换页面内容 |
| 页面模板 | 14个内容导航项对应的页面结构，用于快速构建272个页面 |

## 2. 系统概述

### 2.1 系统简介
DC在线业务系统是一个综合性的业务管理平台，用于实现DC业务的在线管理、监控和数据分析，提供全面的业务功能和良好的用户体验。

### 2.2 系统功能
- 全景俯瞰：提供运营全景、全过程价值链、资源池等视图
- 在线业务：分析访客、渠道、行为、内容等业务数据
- 应用服务：监控流量承载、在线体验、服务传递效率等
- 基础资源：管理存量、服务可用性、使用和转化等资源数据
- 服务健康：提供健康状态报告、黑名单状态、常见问题等信息

### 2.3 系统登录

#### 2.3.1 登录页面
1. 打开浏览器，输入系统URL地址
2. 系统会自动跳转到登录页面

#### 2.3.2 登录步骤
1. 在登录页面输入用户名和密码
2. 点击「登录」按钮
3. 登录成功后，系统会自动跳转到系统首页

#### 2.3.3 忘记密码
1. 在登录页面点击「忘记密码」链接
2. 按照提示输入用户名和邮箱
3. 系统会发送重置密码链接到您的邮箱
4. 点击邮箱中的链接，按照提示重置密码

## 4. 主要功能模块

### 4.1 全景俯瞰

#### 4.1.1 运营全景
- **功能描述**：展示系统整体运营情况的全景视图
- **主要内容**：实时数据概览、关键指标展示、业务趋势分析
- **使用方法**：
  1. 点击左侧导航「全景俯瞰」→「运营全景」
  2. 在内容导航中选择需要查看的内容（如实时动态、近期发生等）
  3. 使用筛选条件调整查看的数据范围
  4. 点击图表或数据进行深入查看

## 5. 常见操作指南

### 5.1 如何查看报警信息
1. 点击顶部导航的「报警」按钮
2. 选择需要查看的报警类型（报警状态、督促、报警处理绩效）
3. 使用筛选条件调整查看的数据范围
4. 查看报警列表和详情
5. 对报警进行处理或关闭操作

### 5.2 如何创建和管理任务
1. 点击顶部导航的「任务」按钮
2. 选择「应急响应任务」、「视情干预计划任务」或「常态化任务」
3. 点击「新增任务」按钮
4. 填写任务信息，包括任务名称、类型、内容、预计完成时间等
5. 点击「保存」按钮创建任务
6. 在任务列表中可以查看、编辑、删除和执行任务

### 5.3 如何查看在线状态
1. 点击顶部导航的「在线」按钮
2. 选择需要查看的在线信息类型（岗位责任情况报告、在线值守报告、岗位责任涉及报告、岗位综合绩效报告）
3. 使用筛选条件调整查看的数据范围
4. 查看在线状态信息

### 5.4 如何管理个人信息
1. 点击顶部导航的「个人中心」按钮
2. 选择「注册信息」、「个性化设置」或「我的内容」
3. 在注册信息页面可以查看和编辑个人基本信息
4. 在个性化设置页面可以设置系统偏好和通知设置
5. 在我的内容页面可以查看和管理个人相关内容

### 5.5 如何导出数据
1. 进入需要导出数据的页面
2. 点击页面上的「导出」按钮
3. 选择导出格式（Excel、PDF等）
4. 系统会生成导出文件并下载到本地

## 6. 常见问题解答

### 6.1 如何重置密码？
1. 在登录页面点击「忘记密码」链接
2. 输入用户名和邮箱
3. 系统会发送重置密码链接到您的邮箱
4. 点击邮箱中的链接，按照提示重置密码

### 6.2 如何修改个人信息？
1. 登录系统
2. 点击顶部导航的「个人中心」按钮
3. 选择「注册信息」
4. 点击「编辑」按钮
5. 修改个人信息
6. 点击「保存」按钮

### 6.3 如何查看历史数据？
1. 进入需要查看历史数据的页面
2. 在筛选区选择时间范围
3. 点击「查询」按钮
4. 系统会显示指定时间范围内的历史数据

### 6.4 如何设置系统通知？
1. 登录系统
2. 点击顶部导航的「个人中心」按钮
3. 选择「个性化设置」
4. 在通知设置中选择需要接收的通知类型
5. 点击「保存」按钮

### 6.5 如何联系系统管理员？
1. 点击顶部导航的「个人中心」按钮
2. 选择「我的内容」
3. 在页面底部找到系统管理员联系方式
4. 或通过公司内部通讯工具联系系统管理员
